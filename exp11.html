Program 1:

                #include<stdio.h>

                void main() {
                    int i, j, n, block[20], start;
                    printf("Enter the no of file: \n");
                    scanf("%d", &n);
                    printf("Enter the number of blocks needed for each file: \n");
                    for (i = 0; i < n; i++)
                        scanf("%d", &block[i]);
                    start = 0;
                    printf("\t\tFile Name\tStart\tSize of file\t\t\n");
                    printf("\n\t\tFile1\t\t%d\t\t%d\n", start, block[0]);
                    for (i = 2; i <= n; i++) {
                        start = start + block[i - 2];
                        printf("\t\tFile%d\t\t%d\t\t%d\n", i, start, block[i - 1]);
                    }
                }

            
              Algorithm:
              -----------
              1. Start the program.
              2. Declare variables:
                  - i, j, n
                  - block (array to store block count)
                  - start (to store starting block)
              3. Prompt user for number of files (n).
              4. Prompt user for blocks needed for each file and store in 'block' array.
              5. Initialize 'start' to 0.
              6. Display table header:
                  - "File Name", "Start", "Size of file"
              7. Display details of first file (File1):
                  - Starting block: 0
                  - Size: block[0]
              8. For each subsequent file (File2 onwards):
                  a. Update 'start' by adding size of previous file (block[i - 2]).
                  b. Display details of file:
                      - Name: Filei
                      - Start position: start
                      - Size: block[i - 1]
              9. End the program.
